weblogic.properties changes for NMC2

Make the following changes in each of the sections listed below (click here to see an example of an editted file)

WEBLOGIC EJB DEMO PROPERTIES

Add: 
 
weblogic.ejb.deploy=d:/nmc2/code/nmc2.jar
where "d:/nmc2" is the root of the code tree

WEBLOGIC JDBC CONNECTION POOL MANAGEMENT

Add: 
weblogic.jdbc.connectionPool.ejbPool=\    
        url=jdbc:weblogic:oracle,\ 
        driver=weblogic.jdbc.oci.Driver,\    
        loginDelaySecs=1,\  initialCapacity=4,\    
        maxCapacity=10,\  capacityIncrement=2,\    
        allowShrinking=true,\  shrinkPeriodMins=15,\    
        refreshMinutes=10,\  testTable=dual,\    
        props=user=techplanner;password=nmc2;server=DevilMountain 

      And add the following after "Set up ACLs for this connection..." 
      
 
        
weblogic.allow.reserve.weblogic.jdbc.connectionPool.ejbPool=everyone      
        weblogic.allow.reset.weblogic.jdbc.connectionPool.ejbPool=everyone 
        weblogic.allow.shrink.weblogic.jdbc.connectionPool.ejbPool=everyone
        
 
      
 
 USER-WRITTEN AND DEMO
        SERVLET REGISTRATIONS
 Add a section to register the NMC2 Servlets
      beginning with the following header: 
      
 
        
###################################################################
        # Registration of NMC2 Servlets.  The initArgs of SpiderLogin # are as
        follows: # #       WebServer  = the URL of
        the WebServer #       ServletURL = suffix to
        WebServer URL to get at servlets #      
        FileURL    = suffix to WebServer URL to get at files #
        ###################################################################
        
 After the above header add the individual servlet
      registrations.  These can be obtained using the
      WLPropertiesDataForNewServlets.xls
      spreadsheet.  This should produce a series of servlet registrations
      (several dozen) which look like: 
  
      
 
        
weblogic.httpd.register.SrvInitialToolSel=com.prn.nmc2.Presentation.SrvInitialToolSel
        weblogic.httpd.initArgs.SrvInitialToolSel=\
               
        WebServer=http://DevilMountain.corp.foliage.com:7001/,ServletURL=,FileURL=file/NMC2/
        
  
      

Cut and paste the results after the header above.
