JDK

Installation

  • JDK 1.2.2 jdk1_2_2-006-win.exe
    Install from Foliage3 or download from Sun at http://java.sun.com/products/jdk/1.2/download-windows.html
    Save and execute the file. (installed into E:\jdk1.2.2).
  • JDK 1.2.2 documentation jdk1_2_2-001-doc.zip.
    Also available from sun at http://java.sun.com/products/jdk/1.2/download-docs.html .
    Save, Unzip the file into the JDK root directory (note that the file paths are already in the Zip file).
  • JDK 1.2.2 sources (optional)jdk1_2_2-src.zip
    Save, Unzip the file to the JDK root directory (note that PART of the file paths are in the Zip file).

Configuration

I suggest avoiding using a system CLASSPATH environment variable. Neither Kawa nor WebLogic require it and the multiple JVMs on the machine can get tangled.
