Oracle Client

Installation

  • Install Oracle Client software from \\Longshot\Oracle8i\setup.exe. (this is very large so use a CD if you have it!)

Configuration

  • Run Oracle Net8 Easy Config Application (Start | Programs | Oracle for Windows NT | Oracle Net 8 Easy Config).

On the east coast:

  • Net Service Name is "PICS".
  • Select TCP/IP network protocol.
  • Specify "StAndrews" as the hostname for the database.
  • Set Database SID to be "PICS"
  • test connection using user=PICS and password=PICS
