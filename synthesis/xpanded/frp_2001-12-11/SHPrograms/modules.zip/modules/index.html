SVG Fundamentals

To introduce our creative department to the exciting new world of Scalable Vector Graphics (SVG), Iguana Studios has created four lesson plans to walk through the fundamentals of the new markup language. Each lesson contains one or more pre-constructed SVG files. These files should can be viewed alongside the instructions in each lesson, and you can also download each lesson as a separate ZIP file.

These files can be opened, read and edited in any text editor. To view the results of any of your edits, drop them into a browser that has the SVG plug-in installed.

[ We recommend editing in BBedit or any other text editor that can give you a line count for the code. Any SVG errors will be displayed in the browser status bar for quick reference in the form: "Syntax error : line 5, column 32"]

  • Lesson Two : Layers and <defs>
