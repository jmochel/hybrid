Class CodeGenerator

java.lang.Object
  |
  +--codegen.CodeGenerator
public class CodeGenerator
extends java.lang.Object

This is a source code generator. It's primary use is to generate classes derived from other classes such as RMI-stubs, object-proxies etc. This is done by writing a code-template using a XML-like language and then subclass this class and implement the tags that have been used in the template file.
